# Projeto Barbearia - Atividade 4

Sistema de agendamentos web desenvolvido com Flask e MySQL.

## 🚀 Como rodar o projeto
1. Clone este repositório ou baixe os arquivos.
2. Instale as dependências: `pip install flask mysql-connector-python python-dotenv`
3. Crie um arquivo `.env` na raiz seguindo o modelo `.env.example`.
4. Importe o arquivo `banco.sql` no seu MySQL Workbench.
5. Execute o comando: `python app.py`
